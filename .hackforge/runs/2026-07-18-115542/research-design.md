# Design Research: Obsidian Memory UI

## Product Class: AI/ML & Dashboard/Analytics
**References:** Linear, Reflect Notes, Notion, Heptabase

### 1. Color and Gradients
- Sleek OLED black bases (`#050508`) combined with high-contrast active borders (`border-white/10` to `border-white/15`).
- Staggered background radial glows using primary purple (`#7c3aed`) and secondary cyan (`#06b6d4`) offsets to create depth without adding heavy assets.

### 2. Layout & Interactions
- Split layout: Sidebar navigation panel (240px wide) + main content workspace grid.
- Bento grids for data metrics and automation rules.
- Fast, fluid viewport movements on knowledge graphs (WebGL & 2D fallback canvas).

### 3. Fonts
- Headings: **Outfit** (geometric, modern, variable weight).
- Body: **Inter** (high legibility, UI-optimized).
- Code/Data: **JetBrains Mono** or system monospaces.
